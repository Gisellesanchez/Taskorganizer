# Task Organizer

This project is a task organizer web-based tracker was built using Django that allows the user to create and view what tasks have been complete or still need to be done. The application stores the tasks in a database and then diplaus them through a fun interactive interface. 

## Instructions for Build and Use

[Software Demo]

Steps to build and/or run the software:

1. First install Python the newest update the better
2. Install Django
3. Navaigate to the project folder
4. Run the development server
5. Open a browser and go to: http://127.0.0.1:8000/

Instructions for using the software:

1. Open the application in your broswer
2. Add a new task using the provided form
3. View the list of tasks displayed on the page with all that needs to be   accomplished.
4. Interact with task by adding, viewing and deleting them.
